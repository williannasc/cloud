## Introdução

Um simples sistema de upload e download de arquivos com controle de acesso, estilo nuvem privada.


## Funcionalidades

- Acesso restrito
- Upload e download de arquivos
- Criação de pastas e subpastas em hierarquia
- Renomeação e exclusão de arquivo
